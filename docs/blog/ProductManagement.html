# 市場

導入期　　顧客に見向きもされない時期
成長前期　顧客からの売上が伸びつづけ、利益率の高い時期
成長後期　売上が伸び続け、著名な大企業が参入しはじめ、利益率が悪くなる時期
成熟期　　購入可能性のある顧客には行き渡り、売上は停滞し、合併や撤退が始まる時期
衰退期　　売上が下がりつづける、プロダクトの寿命が尽きる時期
