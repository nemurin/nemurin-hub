# vuepress

# 初めに

VuePressは軽量なマークアップ技術文書サイト。

# Config

  • テーマが使える
  • アプリレベルの強化
  • TypeScriptでも書ける

# Assets

  • 画像は相対で書くと良い
  • configのaliasも使える
  • publicも使える
  • ベースURL変更も対応

# マークダウン拡張

  • ヘッダーアンカー
  • link
  • frontmatter
  • GitHub形式のtable
  • emojiが使える
  • 目次表　[[(toc)]]
  • タイトル　TIP,WARNING,DANGER,DETAILS
  • シンタックスハイライト　コード記法に言語名追加
  • ラインハイライト　指定行ハイライト
  • ライン番号 configにlineNumbers:trueを追加
  • コードスニペット　ハイライトもあるよ

# Vueの使用

  • vueっぽいv-forもできる
  • vueのようなコンポーネント思考
  • エスケープで灰色にできる
  • sassなどのプリプロセッサも使える

# 国際化

  • pathで言語を切り替えれる

# build

GitHubPagesを僕は使う

# Frontmatter

  • title,lang,layout,permalink,metatitle,正規化URL
  • navbar,sidebar,prev,next,search,tags

# permalink

永続的なリンクのこと

# markdown slot

  • Content slot-keyでマークダウンから挿入可能
