                                          # auth

                                          • sessionに保存する
                                          • hashにpbkdf2-passwordを使用する

                                          • 制限ページはrestrict()をかます,認証だめならloginへ戻す
                                          • logoutはsessionを破棄
                                          • /login(post),/logout,/restrict

                                          • pathによってhtml,text,jsonを使い分ける

                                          # cookie-session
