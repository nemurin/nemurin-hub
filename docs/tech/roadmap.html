                                                # インターネットとは

                                                インターネットという人類の叡智の集積を考える。

                                                • ウェブという網目状に広がったインターネット構造を人類は生み出した。データセンターやラストワンマイル問題がある。

                                                  https://roadmap.sh/guides/what-is-internet (opens new window)
                                                • 世の中はIPアドレスという住所のような番号で管理されている。ISPも管理している。
                                                • DNSという考え方で数字を言語に変換している
                                                • ルーティングという発想で網目を校正している。
                                                • 元々はHTTPという文書送信システムだった。パケットというもので送受信していた。
                                                • ブラウザというものでリッチにHTTPを復元している
                                                • SSLという暗号化標準が導入されている
                                                • SMTP通信というメールの通信方式がある
                                                • TCPというIP解釈方式がある
                                                • 暗号化と公開鍵の考え方がある
